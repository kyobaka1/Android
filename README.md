<h4>Lưu ý khi dùng Git:</h4>
- Trước tiên phải Pull file về, thử chạy cùng với phần của mình trên máy.</br>
- Xem tên file có trùng với các bạn khác hay không?</br>
- Commit và Push phần của mình lên.</br>
